코드 설명!

1. update_FishShaped_Bun = FishShaped_Bun.object.get(id=FishShaped_Bun_id)
update_FishShaped_Bun 는 제가 만든 함수 이름입니다.
FishShaped_Bun 이라는 클래스에서 object(객체)를 get(id=붕어빵의 아이디값(=붕어빵마다 id가 다름))받아오는것

2. update_FishShaped_Bun.name = request.POST['name']
update_FishShaped_Bun.name: 마찬가지로 임의로 작성한 함수이름
request.POST['name'] : request(요청한다) POST방식을사용해서'name'받는다

3. if request.FILES.get('FishShaped_Bun_img')
만약에 이미지를 get 방식으로 받았을경우 update_FishShaped_Bun.img = request.FILES.get('img')
img파일로 파일을 받은것을 update_FishShaped_Bun.img에 넣어준다.
4.return redirect('/FishShaped_Bun/') + str(update_FishShaped_Bun.id)
URL 이동함수입니다. 기본적인 /FishShaped_Bun/에서 str(붕어빵의 고유 id 값) 예를들어서 팥 붕어빵의 id값이 1이고 슈크림 붕어빵의 id값이 2이면 www.~~~.com/FishShaped_Bun/1 : 팥 붕어빵에 대한 페이지 www.~~~.com/FishShaped_Bun/2 : 슈크림 붕어빵에 대한 페이지

MTV 패턴

Model / Template / View

Model : DB를 다루는 역할 -사용자의 요청으로 Model과 DB에서 데이터를 받아온다
Model : 데이터 찾기 담당
Template : 보여지기 담당
View : 처리 담당
MTV 패턴 : Model , Template , View가 독립적으로 임무수행을 통해 서버를 구동하는 패턴
view로 데이터를 가져오고 Template으로 보여주는 개념

핵심은 View
